# 音声入力機能ガイド

業務支援システムv4.0では、3つの音声入力方法をサポートしています。

## 🪟 Windows音声入力 (推奨)

### 特徴
- Windows標準の音声入力機能（Win+H）を利用
- 高精度で安定した音声認識
- 追加ライブラリ不要
- 日本語に最適化済み

### 使い方
1. 音声入力ボタンをクリック（デフォルト動作）
2. または音声入力ボタンの▼メニューから「Windows音声入力 (Win+H)」を選択
3. Windows音声入力が起動します
4. マイクアイコンが表示されたら発話してください

### ショートカット
- `Ctrl+H`: 直接Windows音声入力を起動

### 対応OS
- Windows 10 (May 2019 Update以降)
- Windows 11

## 🎙️ アプリ内音声認識

### 特徴
- アプリケーション内蔵の音声認識
- Google Speech Recognition APIを使用
- リアルタイム音声認識

### 使い方
1. 音声入力ボタンの▼メニューから「アプリ内音声認識」を選択
2. マイクへのアクセス許可を与える
3. ボタンが「聞き取り中...」に変わったら発話

### 必要なライブラリ
```bash
pip install SpeechRecognition pyaudio
```

### ショートカット
- `Ctrl+Shift+V`: アプリ内音声認識を開始

## ⌨️ テキスト入力（フォールバック）

### 特徴
- 音声認識が利用できない場合の代替手段
- キーボードで音声コマンドを入力
- 全環境で利用可能

### 使い方
1. 音声入力ボタンの▼メニューから「テキスト入力」を選択
2. ダイアログに音声コマンドをテキストで入力

## 対応音声コマンド例

### 取引先管理
- "新規取引先"、"新しい取引先" → 新規取引先登録画面
- "取引先検索"、"探す" → 取引先検索画面

### 商品管理
- "新規商品"、"商品登録" → 新規商品登録画面
- "在庫確認"、"在庫調整" → 在庫調整画面

### データ出力
- "エクスポート"、"出力" → データエクスポート

## トラブルシューティング

### Windows音声入力が起動しない場合
1. Windows設定 → プライバシー → 音声認識でマイクアクセスを有効化
2. 言語設定で日本語音声認識がインストールされていることを確認

### アプリ内音声認識が動作しない場合
1. マイクがWindowsで認識されているか確認
2. 必要なライブラリがインストールされているか確認
3. ファイアウォールでインターネット接続が許可されているか確認

### 音声認識精度を向上させるコツ
- 静かな環境で使用する
- マイクに近づいて明瞭に発話する
- 短いフレーズで区切って話す
- 専門用語は避け、一般的な表現を使用する

## 設定

音声入力機能の有効/無効は、システム設定画面で変更できます：
- 設定 → AI機能 → 音声認識機能
